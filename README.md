# Python-Portfolio
AP Computer Science Principles work

The grocery program implements data abstraction by managing two parallel arrays that link item names directly to their corresponding prices. Users input a specific budget, triggering a filtering algorithm that traverses the datasets to isolate affordable options. The system optimizes the shopping process by dynamically displaying only the items that meet the user's financial criteria.

The simulation program models the classic fable by executing a randomized simulation of a race between a tortoise and a hare. The program utilizes conditional logic and loops to update the position variables of each competitor based on distinct movement probabilities. It demonstrates complex algorithmic behavior by tracking the race progression until a winning condition is met.

The slots application replicates a functional digital slot machine by processing user bets and triggering a randomized reel selection. The underlying algorithm utilizes a random number generator to determine symbol outcomes across multiple independent variables. It then evaluates these outcomes using nested conditional statements to calculate payouts and update the user's total points.

This to do list program is a productivity tool that manages daily tasks through a dynamic user interface that tracks project completion. The program accepts text input to populate a list and utilizes event handlers to detect when a user checks off a task. It updates the state of the data structure in real time, providing immediate visual feedback to the viewer.

The word counter program processes a string of user input to calculate and output the exact number of words it contains. The core algorithm iterates through the string character by character, utilizing conditional logic to identify spaces and punctuation as word boundaries. It provides a foundational example of string manipulation and sequential data processing in a user-facing application.
